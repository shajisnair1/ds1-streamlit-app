## 7.29.5.2 Gage Calibration

Pressure measuring devices shall be calibrated to maintain $\pm 2\%$ accuracy of full-scale range. Pressure gauge must be able to read at least $150\%$ of the test pressure but not more than $500\%$ of the test pressure. See section 1.7 for calibration requirements for the pressure and airflow gages used for function testing.

## 7.29.6 Post-Qualification Requirements

Clean and dry the threaded connections. Visually inspect thread and seal surfaces in accordance with paragraphs 7.14.5.2 and 7.14.5.7 to ensure damage did not occur from the test plugs during function test. Apply appropriate thread compound. Install thread protectors. Place a 2-inch wide ($\pm 1/4$ inch) white paint band around an acceptable component. The paint band should be 6 inches $\pm 1$ inch from the pin shoulder. The paint band should be 12 inches $\pm 2$ inches from the box shoulder for box by box components. Using a permanent paint marker on the outer surface of the tool, write or stencil applicable DS-1 qualification class, the date, and the name of the company performing the inspection.

## 7.29.7 Storage Requirements

- Before storing an air hammer ensure it has been thoroughly oiled.
- After oiling, both ends of the air hammer shall be covered to keep the internal components clean and moisture free.
- Air hammers shall be stored and transported in a horizontal orientation to prevent oil starvation.
- Air hammers shall be used within 90 days or returned to a shop for disassembly, inspection, and rebuild.

## 7.30 Shop Makeup Procedure

### 7.30.1 Scope

This procedure covers the proper application of optimal makeup torques (MUT) and verification of bucked on assemblies shipped to rigs to ensure connections are made up correctly.

### 7.30.2 Determining Optimal MUT

a. Drilling and Completion Applications: The MUT applied to API Rotary-Shouldered Connections (RSCs) to be used for drilling applications shall be the API recommended MUT as defined in API RP 7G unless specifically requested otherwise by the customer. API MUT values can be calculated using various programs, including T H Hill Associates' Drillstring Toolbox™ app. Some bevel diameters specified by DS-1 Volume 1 may result in seal stresses that exceed the yield strength of the material when API MUT is applied (see paragraph 7.30.2g).

b. Landing Applications: The MUT applied to API RSCs to be used for landing applications shall be based on the lower of API MUT or T4 values as defined in API RP 7G. T4 MUT values provide the maximum tensile capacity without shoulder separation. T4 MUTs can be calculated using various programs, including T H Hill Associates' Drillstring Toolbox. Some bevel diameters specified by DS-1 Volume 1 may result in seal stresses that exceed the SMYS when T4 MUT is applied (see paragraph 7.30.2g).

c. Proprietary Connections: MUTs and makeup requirements for proprietary connections shall be provided by the manufacturer. DS-1 Volume 2 contains several tables of recommended makeup torque values, but these must be verified with the manufacturer's recommendations. When used in a landing string application, the equipment provider must verify that the required tensile capacity is fit for purpose.

d. OCTG Connections: MUTs applied to proprietary OCTG connections shall be obtained from the manufacturer and applied with continuous-turn equipment using the torque-turn method. If required by the customer, a licensed thread representative may be present when making up proprietary connections. API OCTG connections shall be made up to the API nominal power-tight position.

e. Fit-for-Purpose MUTs: In the event where a MUT to be applied does not meet the criteria listed above, a fit-for-purpose technical justification must be provided. The written technical justification should at a minimum include the tensile and torsional capacity as well as the estimated tensile and torsional loads
