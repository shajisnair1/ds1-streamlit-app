**Component Record.** A document that confirms to the assembler that a component is ready to be used in the assembly of a tool. Examples may be: An inspection report, traceability record, use record, or other record. Component records typically accompany components that are used over and over, and that require inspection or maintenance between uses. Although every component must have a part number, not all components will be accompanied by component records.

**Configuration.** A designation that reveals how a specialty tool is trimmed or dressed. For example, a nominal 8-inch tool may be configured with either 6-5/8 Regular or NC56 end connections.

**Covered Tool.** A specialty tool that is ordered and delivered under the requirements of this standard.

**Critical Buckling Load.** The amount of mechanical compression a string section or specialty tool can carry without buckling.

**Critical Dimension.** A dimension on a specialty tool that would normally be of interest to rig personnel. This is assumed to be limited to data required for efficiently operating the tool or operating the drill string with the tool in place. Critical dimensions may include tool length, tool outside diameter, fishing neck length and (if applicable) nominal inside diameter, drift diameter, and retracted and expanded outside diameters. Other dimensions may be deemed critical by the customer or manufacturer.

**Customer.** The party that is at immediate economic risk in the event of a specialty tool failure. Except in a turnkey drilling situation, this will normally be an operating company that is drilling a well.

**Customer's Representative.** A person who may be sent to the vendor's location to witness inspection, assembly, testing, or some other operation on a specialty tool or tool component.

**Design Factor (or Safety Factor).** As employed by a specialty tool designer, a number equal to or greater than 1.0 that is applied to design equations to account for lack of precision in the equation and/or for other factors about which the designer may not be certain.

**Dressing a Tool.** Also called "trimming." A process used during tool assembly that allows tools of the same model number and part number to function differently from one another. For example, the conditions under which a tool will activate may be varied by varying the properties of shear pins used during tool assembly, or by varying some adjustable part on or in the tool.

**Drill String Designer.** The person who configures a drill string that contains a specialty tool.

**End (Rig End) Connection.** A connection in the load path of a specialty tool that carries string loads, and that is the end connection that attaches the tool to the drill string or casing string.

**Expendable Component.** A component that is always replaced with a new component upon disassembly and reassembly of a specialty tool.

**Externally Upset Tubing.** Tubing with two integral pin connections, one of which may already have a coupling applied during manufacturing, which is externally upset near both connections creating larger ODs near the connections.

**Fatigue.** Progressive, localized permanent structural damage that can occur at high stress points in a specialty tool component when the tool undergoes cyclic loading.

**Function (noun).** The name or description of an operation that a specialty tool is designed to perform. In this standard, the word is used only as a noun or an adjective, as in "function testing" below.

**Function Test Procedure.** A written document that details how the tester is to conduct a function test.

**Function Test Report.** A summary giving the tool description and serial number and describing the post-assembly function tests performed on the tool.

**Function Testing.** Simulating the exercise of as many of the functions of a specialty tool as practical, after assembly but before the tool leaves the vendor's shop.

**Functional Failure.** The failure of a specialty tool to perform its intended function. Functional failure does not involve the loss of any part of the specialty tool or the drill string below the specialty tool (if applicable).
