## 5.4 Function Test Report

The results of the Function Test Procedure shall be documented in a Function Test Report. The test results will follow the same order and numbering as the Function Test Procedure. The Function Test Report must document the:

- Function Test Procedure and revision used.
- Job Order used.
- Date of testing performed.
- Part number and serial number for the tool or sub-tool tested.
- Satisfactory completion of all steps required for testing.
- Results of every test performed, with a clear "pass" or "fail" label. (A Function Test Report is required to document the testing even if the tool or sub-tool fails the test and is re-routed for repair work.)
- Tester's name and signature.
