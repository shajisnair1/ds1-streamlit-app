## 7.16.3 Preparation

a. All part numbers and serial numbers shall be recorded on all reports.

b. Connections shall be clean so that no scale, mud, or lubricant can be wiped from the thread or shoulder surfaces with a clean rag.

## 7.16.4 Procedure and Acceptance Criteria for API and Similar Non-Proprietary Connections

It is presumed that a Visual Connection Inspection (section 7.14) will be performed in conjunction with this inspection. If the Visual Connection Inspection (section 7.14) will not be performed, steps 7.14.4, 7.14.5.6, 7.14.5.9, and 7.14.5.10 shall be added to this procedure.

a. Box Outside Diameter (OD). The OD of the box connection shall be measure 4 inches, $\pm 1/4$ inch from the shoulder. At least two measurements shall be taken spaced at intervals of 90 degrees $\pm 10$ degrees. The OD shall meet one of the following criteria, as applicable:

- The manufacturer's acceptance criteria for OD.
- For components connected to HWDP, the OD must meet requirements listed in Table 7.38.
- For components connected to drill collars or drill-collar-like components, the box OD (in combination with pin ID) shall result in a BSR within the customer's specified range. Dimensions for commonly specified BSR ranges are given on Table 7.37. BSR values for various connection types and sizes are provided in Table 7.50.

If a conflict arises, the manufacturer's criteria shall apply.

b. Pin Inside Diameter (ID). The pin ID shall be measured under the last thread nearest the shoulder, $\pm 1/4$ inch. The ID shall meet one of the following criteria as applicable:

- The manufacturer's acceptance criteria for ID.
- For components connected to HWDP, the ID must meet requirements listed in Table 7.38.
- For components connected to drill collars or drill-collar-like components, the pin ID (in combination with box OD) shall result in a BSR within the customer's specified range. Dimensions for commonly specified BSR ranges are given on Table 7.37. BSR values for various connection types and sizes are provided in Table 7.50.

If a conflict arises, the manufacturer's criteria shall apply.

c. Box Counterbore Diameter. The box counterbore diameter shall be measured as near as possible to the shoulder (but excluding any ID bevel or rolled metal) at diameters 90 degrees $\pm 10$ degrees apart. The counterbore diameter shall not exceed the maximum counterbore dimension listed in Table 7.37 or 7.38, as applicable.

d. Box Counterbore Depth. The counterbore depth shall be measured (including any ID bevel). Counterbore depth shall not be less than the value shown in Table 7.37.

e. Pin Stress Relief Groove. Unless waived by the customer, all API connections NC38 and larger that make up to BHA components, including API connections of saver subs that make up to a BHA specialty tool, shall be equipped with pin stress relief grooves or they shall be rejected. The diameter and width of the API pin stress relief groove shall be measured and shall meet the requirements of Table 7.37 or 7.38, as applicable. The pin stress relief groove length shall be measured from the connection shoulder to the first full thread by placing a metal rule on the thread taper and squared against the connection shoulder, as depicted in Figures 7.51, 7.52 and 7.53. "First full thread" is defined as the thread that is closest to the pin shoulder and reaches the same height and thread profile as the second thread. The location of the first full thread can be identified by rotating the profile gauge until the absolute minimum amount of light is visible between the thread form and the profile gauge.

f. Box Boreback.
