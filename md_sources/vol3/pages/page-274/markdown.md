# Table 3.12.1 Float Bore Calculation

(for cases not covered in Table 3.12)

## To determine Float Bore Depth (A):

1) Measure Float Bore Diameter (R)
2) Obtain Valve Size and Valve Length (Lₐ) corresponding to (R) from Table 3.12 and the Max Pin Length (Lₐ) for the Sub's connection from Table 3.9.
3) Float Bore Depth

Box Down: (A) ± 1/16" = Lₐ + Lₐ + 3/16"

## Where:

R = Bore Diameter
Lₐ = Valve Length
Lₐ = Max Pin Length (Table 3.9)
3/16" = Margin for proper Pin and Valve separation
A = Bore Depth

Box Up: (A) ± 1/16" = Lₐ + Lₐ + 3/16" + minimum length needed to have all seals seated within the float bore

1. See DS-1, Volume 3, Section A.6 in the Appendix for more information on calculating float bore depth for box up configurations.
