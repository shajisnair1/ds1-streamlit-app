Maintenance Class (or Classification). A designation that indicates the level of maintenance that a specialty tool or sub-tool received before it was shipped.

Makeup Torque. The torque to which threaded connections (including rotary shouldered connections) in the tool will be screwed together during assembly.

Manufacturer. The company that is responsible for the design and manufacture of a specialty tool. It may also be called the Original Equipment Manufacturer (OEM).

Manufacturer's Authorized Representative. A person who may be sent to the rig instruct the customer or to witness or manage the customer's use of a specialty tool.

Material Specification. A document that specifies the material properties that a component must possess in order for the specialty tool in which the component is used to meet its specified load rating.

Material Test Report (MTR). A document that gives material testing results traceable to an identifiable component or components. The MTR is prepared by a material manufacturer or material testing facility.

Mechanical Compression. The axial compressive force in a drill string section or specialty tool, neglecting pressure area forces.

Midbody Connection. A connection in the load path of a specialty tool that carries loads, and that is not one of the end connections that attach the tool to the drill string or casing string.

Misuse. Loading a tool beyond its rated load capacity.

Model (Tool Model). A designation applied to a set of specialty tools that, for a given size and configuration, utilize the same components, part numbers, assembly procedures, testing procedures, and inspection procedures.

# N

Neutral Point. A point in a drill string immediately above which the string is stable, and immediately below which the drill string is buckled. Except in a vertical hole, the neutral point and the point of zero tension are never the same. A drill string may have more than one neutral point at the same time.

New Component. A component that has never been assembled into and used in a specialty tool.

Nonconforming Component. A component that, during inspection, fails to meet the required acceptance criteria.

Non-Upset Tubing. Tubing with two integral pin connections, one of which may already have a coupling applied during manufacturing, which has no external upsets giving it a continuous and unchanging OD along the entire length of the joint.

Normal Weight Drill Pipe (NWDP). Any size or weight of drill pipe that is listed on Table 7.4 of this Standard.

# O

Operating Constraint. A limit that is established by a tool vendor and placed on the operation of its tool. Operating the tool outside this constraint is considered detrimental for reliable tool operation.

Overload. A condition in which the load applied to a specialty tool exceeds the load rating of that tool.

# P

Part Number. A number that identifies one type of component from other (dissimilar) components. Identical components will have the same part number.

# R

Rated Load Capacity. See "Load Rating."

# S

Safety Factor. See "Design Factor."

Serial Number. A unique identifier assigned to a specific serialized component to differentiate it from other components of the same part number. Identical serialized components will have the same part number but different serial numbers.

Serialized Component. A component that is assigned a serial number.

Significant Dimension. A dimension on a specialty tool that would normally be of interest to rig personnel.

SMYS. An acronym that designates the manufacturer's specified minimum yield strength for the material in a component.
