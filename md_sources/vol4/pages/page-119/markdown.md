## 7.22.5 Post Cleaning

Post-inspection cleaning is necessary where magnetic particles could interfere with subsequent processing or with service requirements. Suitable post-inspection cleaning techniques shall be used which shall not interfere with subsequent requirements. The white coating paint applied during inspection shall be stripped from the pipe surface in order to facilitate post inspection markings.

## 7.23 Specific Requirements for Shop Qualification of Sub-Surface Safety Valves

### 7.23.1 Scope

This section provides additional specific requirements for shop inspection and assembly of Sub-Surface Safety Valves (SSSV).

### 7.23.2 Requirements from API Specifications

Three specifications developed by API already exist regarding sub-surface safety valves. The procedures, parameters and acceptance criteria included in the latest revisions of these API specifications shall be followed during the inspection, assembly, and testing processes, as applicable. In addition to any applicable requirements from the API specifications, the requirements included in Section 7.23.3, 7.23.4, and 7.23.5 shall also be followed. A summary of the information from each API specification is included below.

#### 7.23.2.1 API Specification 14A

API Spec 14A is intended to provide guidelines for the design, qualification, and repair of SSSVs and includes minimum requirements with which suppliers and manufacturers must comply. It also includes recommendations for additional requirements, which are not mandatory.

#### 7.23.2.2 API Specification 14B

API Spec 14B includes requirements for the SSSV system design, installation, operation, testing, redress, support activities, documentation, and failure reporting. Redress only pertains to the replacement of seals which do not require disassembly of the body joint connections. If disassembly is required, then the process is classified as repair and is included in API Spec 14A.

#### 7.23.2.3 API Specification 14L

API Spec 14L provides requirements for landing nipples and lock mandrels within the production/injection conduit for the installation of flow control and other equipment. This also includes requirements for the interface connections to the flow control equipment, along with other completions equipment as well as testing requirements for landing nipples and lock mandrels.

### 7.23.3 Inspection Requirements

The Inspection Program developed by the vendor for the inspection of sub-surface safety valves shall include the following procedural requirements and common inspection methods.

#### 7.23.3.1 Apparatus

A paint marker, a calibrated pit depth gauge, a light capable of illuminating the entire internal surface of the valve, a metal ruler, a tape measure, a flat file or disk grinder, and a borescope are required for the inspection. See section 1.7 for calibration requirements for the pit depth gauge. All apparatus requirements specified in the applicable common inspection methods are also required.

#### 7.23.3.2 Preparation

Record the serial number and tool description. Reject the tool if no serial number can be located unless the customer waives this requirement.

#### 7.23.3.3 Common Inspection Methods Required

- Visual API Round Connection Inspection (7.25)
- Dimensional API Round Connection Inspection (7.26)
- Drift Testing (7.27)
- Blacklight Connection Inspection (7.17) for connections made from ferromagnetic material
- Liquid Penetrant Inspection (7.18) for tool components made from nonmagnetic material
