## 7.4.9 Residual Magnetic Particle Inspection

This procedure covers inspection of ferromagnetic surfaces on which an active field cannot practically be used. The purpose of this procedure is to detect transverse, longitudinal and oblique flaws using either the wet fluorescent residual magnetic particle technique or the dry visible residual magnetic particle technique.

## 7.4.10 UT Connection

This procedure covers examination of rotary-shouldered midbody and end connections for transverse flaws using the ultrasonic compression wave technique.

## 7.4.11 Wet Visible Contrast Inspection

This procedure covers inspection of external surfaces for transverse and three-dimensional flaws, using the wet visible contrast technique with an active AC field or with an active DC field.

## 7.4.12 Visual API Round Connection Inspection

This procedure covers visual examination of new and used API round connections typically found on completion equipment that are compatible with tubing connections to evaluate the condition of the connections.

## 7.4.13 Dimensional API Round Connection Inspection

This procedure covers dimensional examination of new and used API round connections typically found on completion equipment that are compatible with tubing connections to evaluate the condition of the connections.

## 7.4.14 Drift Testing

This procedure covers the drifting of a component to detect any reduction in the ID.

## 7.4.15 Ultrasonic Wall Thickness Inspection

This procedure covers the ultrasonic measurement of wall thickness using compression waves.

## 7.5 Specific Requirements for Shop Qualification of Drilling Jars

### 7.5.1 Scope

This section provides additional specific requirements for shop inspection, assembly, and function testing of drilling jars.

### 7.5.2 Inspection Requirements

The Inspection Program developed by the vendor for inspection of drilling jars must include the following procedural requirements and common inspection methods.

### 7.5.2.1 Apparatus

The following equipment must be available for inspection: paint marker, pit gage, a light capable of illuminating the entire internal surface, metal scale, tape measure, flat file or disk grinder. All apparatus requirements specified in the applicable common inspection methods are also required.

### 7.5.2.2 Preparation

Record the tool serial number and tool description. Reject the tool if no serial number can be located unless the customer waives this requirement.

### 7.5.2.3 Common Inspection Methods Required

- Sub Inspection (7.12)
- Visual Connection Inspection (7.14)
- Dimensional 3 Inspection (7.16)
- Blacklight Connection Inspection (7.17)
- MPI Body Inspection (7.19)

### 7.5.2.4 MPI Body Inspection Coverage

When performing the MPI Body Inspection (section 7.19) common inspection method, the inspection shall cover the entire outside surface of the tool from shoulder to shoulder, and all load bearing components. Pay special attention to areas around load adjustment ports and external oil plug holes that may act as stress risers. Any crack is cause for rejection.

### 7.5.2.5 Visual Body and Internal Hardware Inspection

a. Visually examine the tool case components for mechanical damage. Any cut, gouge, or similar imperfection deeper than 10% of the adjacent wall shall be cause for rejection. Remove any loose scale and raised metal before measuring imperfection depth.

b. Pitting, erosion, cuts, and gouges to internal and external chrome plated sealing areas, or flaking, spalled, or peeling chrome shall be cause for rejection.

### 7.5.3 Assembly Requirements

The Assembly Procedure developed by the vendor for drilling jars must include the following items:

#### 7.5.3.1 Seals

Replace old O-rings and soft seals with new ones prior to reassembly.
