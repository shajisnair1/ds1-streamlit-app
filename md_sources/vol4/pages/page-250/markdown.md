Table 7.49 Float Bore Calculation
(for cases not covered in Table 7.48)

To determine Float Bore Depth (A):

1) Measure Float Bore Diameter (R)
2) Obtain Valve Size and Valve Length (Lp) corresponding to (R) from Table 7.48 and the Max Pin Length (Lp) for the Sub's connection from Table 7.37.
3) Float Bore Depth

Box Down: (A) ± 1/16" = Lp + Lp + 3/16"

Bore Depth

Bore Dwn: (A) ± 1/16" = Lp + Lp + 3/16" + minimum length needed to have all seals seated within the float bore¹

1 See DS-1 Volume 3, Section A.6 in the Appendix for more information on calculating float bore depth for box up configurations.

Where:

R = Bore Diameter
Lp = Valve Length
Lp = Max Pin Length (Table 7.37)
3/16" = Margin for proper Pin and Valve separation
A = Bore Depth
