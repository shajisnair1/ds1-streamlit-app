2. The minimum ID of Premium Class for most RSL types are based on the sin rate, pipe ID, and torsional capacity (TSR not less than 0.8).
3. The minimum OD of Premium Class is based on ID max and torsional capacity (TSR not less than 0.8). If the total joint ID is smaller than ID max, the OD of Premium Class can be even smaller than stated above.
4. The minimum long space excludes hardbanding which is from seat shoulder to the edge of hardbanding.
5. If the actual box OD is less than or equal to the specified bevel diameter plus 4/32 inch, then the bevel diameter range is void and a 1/4" x 45" break edge is required.
6. When conflicts arise between this standard and manufacturer's requirements, the manufacturer's requirements shall apply.
